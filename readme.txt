== Description ==

With this plugin you can get posts and change URL dynamic based on post section by using this Shortcode [get-posts-react]

== Installation ==

1. Upload `load-wordpress-posts-in-react.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php echo do_shortcode('[get-posts-react]'); ?>` in your templates or add [get-posts-react] in page
